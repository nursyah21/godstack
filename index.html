God Stack adalah pendekatan no-build step untuk membangun aplikasi web. Kamu tidak butuh Webpack, Vite, atau NPM. Cukup satu file HTML untuk menangani:

  • DOM Wrapper: Manipulasi elemen ala jQuery menggunakan $().
  • Routing: Client-side routing berbasis hashchange.
  • Reactive State: Sinkronisasi data ke UI secara otomatis.
  • Supabase Integration: Autentikasi dan Database Realtime.

  1. Konfigurasi Supabase:

    Buka framework.html dan cari variabel di bawah ini. Ganti dengan kredensial dari dashboard Supabase-mu:

    const SB_URL = "https://PROJECT_ID.supabase.co";
const SB_KEY = "YOUR_ANON_KEY";
  2. Siapkan Database:

    Jalankan query SQL berikut di SQL Editor Supabase untuk membuat tabel dan kebijakan keamanan (RLS):

    CREATE TABLE items (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  nama_item TEXT NOT NULL,
  user_id uuid REFERENCES auth.users DEFAULT auth.uid()
);

ALTER TABLE items ENABLE ROW LEVEL SECURITY;
-- Tambahkan policy SELECT, INSERT, UPDATE sesuai di file framework.html

2. State Management

Setiap kali store.items berubah, elemen #list-container akan otomatis di-render ulang.
